# DOOMSZZ

Statička web stranica izrađena u HTML-u i CSS-u za Društvo oboljelih od multiple skleroze Zadarske županije. Projekt je osmišljen kao responzivna višestranična stranica s naglaskom na preglednost, pristupačnost i jasno organiziran sadržaj.

## Stranice

- `index.html` - naslovna stranica s uvodnim informacijama i pregledom djelovanja društva
- `about.html` - stranica o društvu, misiji, ciljevima i vrijednostima
- `asistencija.html` - stranica o programu osobne asistencije i podršci korisnicima
- `kontakt.html` - kontakt stranica s osnovnim podacima i kontakt formom

## Kako pokrenuti projekt

Projekt se može pokrenuti na dva načina:

1. Otvoriti `index.html` datoteku direktno u internetskom pregledniku
2. Pokrenuti projekt pomoću Live Server ekstenzije u Visual Studio Codeu

## Tehnologije

- HTML5
- CSS3
- Flexbox
- CSS Grid
- CSS varijable

## Funkcionalnosti

- responzivan dizajn za desktop i mobilne uređaje
- hamburger navigacija u mobilnoj verziji
- semantički HTML elementi
- kontakt forma s HTML validacijom
- SEO meta tagovi
- osnovni elementi pristupačnosti, uključujući skip link i vidljiv fokus
